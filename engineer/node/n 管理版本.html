# n 管理版本

# 安装

npm install -g n

如若安装失败，在该指令后加上 --force，重新安装。

# 查看版本

n ls

# 安装版本

// 安装最新的长期稳定版
n lts

// 安装最新版本
n latest

// 安装指定版本
n <version>

# 删除版本

n rm <version>

# 切换版本

n

上下切换，回车选定版本：

  6.9.4
ο 7.4.0
  4.4.4
