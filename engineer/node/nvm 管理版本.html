# nvm 管理版本

# 安装

下载地址：https://github.com/coreybutler/nvm-windows/releases

选择 nvm-setup.zip 下载安装

# 查看版本

nvm ls

# 安装版本

// 安装最新的长期稳定版
nvm install --lts

// 安装最新版本
nvm install node

// 安装指定版本
nvm install <version>

# 删除版本

nvm uninstall <version>

# 切换版本

nvm use <version>
