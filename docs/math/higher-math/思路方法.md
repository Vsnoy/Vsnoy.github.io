# 思路方法

- 求极限
  - 四则运算法
  - 两个重要极限（$\lim \limits_{x \to 0} \dfrac{sinx}{x} = 1，\lim \limits_{x \to \infty}(1 + \dfrac{1}{x})^{x} = e$）
  - 洛必达法则
  - 等价无穷小替换 （常与换元法联用）
  - 夹逼定理
  - 单调有界数列准则（常与不等式法或数学归纳法联用）
  - 抓大头法（分子分母同除高次幂）
  - 取对数法（$a^{b} = e^{lna^{b}} = e^{blna}$）（常用于幂指函数）
- 求不定积分
  - 凑微分法（换元、积分、还原）
  - 第二类换元法
  - 分部积分法
