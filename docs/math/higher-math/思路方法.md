# 思路方法

- 求极限
  - 直接代入法
  - 四则运算法
  - 两个重要极限（$\lim \limits_{x \to 0} \dfrac{sinx}{x} = 1，\lim \limits_{x \to \infty}(1 + \dfrac{1}{x})^{x} = e$）
  - 等价无穷小替换 （常与换元法联用。有时没有明显的等价无穷小结构，需要自己凑出来，比如先减再加）
  - 洛必达法则
  - 泰勒公式
  - 夹逼定理
  - 单调有界数列准则（常与不等式法或数学归纳法联用）
  - 幂指函数取对数（$a^{b} = e^{lna^{b}} = e^{blna}$）
  - 抽出无穷因子
  - 分子分母同除高次幂
  - 分子有理化
  - 通分
- 求不定积分
  - 凑微分法（换元、积分、还原）
  - 第二类换元法
  - 分部积分法
