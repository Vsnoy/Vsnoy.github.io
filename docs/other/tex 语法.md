# Tex 语法

```
- 对数：\log_{}{} | \ln{} | \lg{}
- 上下角标：^{} | _{}
- 空格：\space | \quad
- 极限：\lim | \limits_{}
- 无穷：\infty
```

```
- 分号：\dfrac{}{}
- 根号：\sqrt{} | \sqrt[]{}
- 加减号：\pm | \mp
- 近似号：\sim
- 双向等号：\Leftrightarrow | \Longleftrightarrow | \Updownarrow
- 小于等于号：\le
- 大于等于号：\ge
- 远小于号：\ll
- 远大于号：\gg
- 省略号：\dots | \cdots | \vdots | \ddots
- 分组大括号：\begin{cases} \end{cases}
- 任意号：\forall
- 存在号：\exists
```
